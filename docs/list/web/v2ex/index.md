脚本名：`sudojia_v2ex.js`

[【前往官网】](https://www.v2ex.com/)

## 奖励说明

每日签到获得货币

## 变量说明

|      Env      | Value  |          Remark           |
| :-----------: | :----: | :---------------------: |
| `V2EX_COOKIE` | Cookie | 多账号用 `&` 或换行隔开 |

## 获取说明

按 F12，选择 Network 网络，然后请求选择 `Doc`，把 Cookie 全部复制，没有出现 `v2ex.com` 那就 F5 刷新一下
