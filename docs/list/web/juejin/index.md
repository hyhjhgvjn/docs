2. 每日使用免费抽奖次数抽奖，幸运值满 6000 必出实物
3. 升级任务：成长等级的社区活跃任务！

## 变量说明

|       Env       |      Value      |         Remark          |
| :-------------: | :-------------: | :---------------------: |
| `JUEJIN_COOKIE` | 稀土掘金 Cookie | 多账号用 `&` 或换行隔开 |

## 获取说明

打开[掘金社区](https://juejin.cn/) 按 F12 -> 选择 Application -> 点击 Cookies -> 只要 `sessionid` 的值
